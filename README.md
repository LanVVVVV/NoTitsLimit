# NoTitsLimit

NoTitsLimit is a mod for **Monster Black Market** —— Removes age tits limit.

### Features

* Children are no longer forced to have TitsType = 0 and may possess non‑zero breast types.
* If a character is a child with a non‑zero breast type, their maximum milk capacity is reduced by half.
